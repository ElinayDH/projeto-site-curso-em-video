<h1>Cursos Grátis</h1>

Curso de HTML5 com CSS3

HTML é a linguagem de marcação com a qual você envolve o conteúdo, para instruir os navegadores sobre cabeçalhos, listas, tabelas, etc. CSS é a linguagem de estilos que você usa para estilizar a página, para instruir os navegadores a alterar a cor, a fonte, o layout e muito mais.

Curso de JavaScript

O JS ou JavaScript é uma linguagem de programação de alto-nível, criada no meio da década de 90, mais precisamente em 1996 pelo lendário programador Brendan Eich que, além de criar o JavaScript, foi também um dos fundadores da Mozilla Corporation.
